APIs.json Format Definitions

The format is designed for public deployment and for consumption by automated software agents (robots). This document is written in loose RFC style for the purposes of clarity, but it is not an official recommendation. Discussion around the format takes place on the APIs.json mailing list, and you can submit any issues via the Github issue management for the project.
