APIs.json Discovery Format

What is APIs.json?

APIs.json is a machine readable approach that API providers can use to describe their API operations, similar to how web sites are described using sitemap.xml.

Contributing

If you have comments on the format, search or anything related, discussion is here on the APIs.json Google Group, and the roadmap is managed using the Github issue management for this repository.

Who is doing this

This effort was started by 3scale and API Evangelist and we’ve already had a lot of great feedback which we’ll reflect in future versions.
